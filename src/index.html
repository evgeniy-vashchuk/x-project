Starter kit for fast HTML / CSS / JS pages layout based on Gulp

X-Project - is starter template with basic tasks for development, which uses Gulp to compile SASS to CSS (with SCSS syntax), add vendor prefixes, group media queries, remove comments and minify CSS, convert ES6 into ES5 using Babel, optimising images (with creating of webp versions and lazyloading), Browsersync for live-reload while development, favicon generator and project archiving.
